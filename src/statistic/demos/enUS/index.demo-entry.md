## Props

| Name  | Type     | Default     | Description                         |
| ----- | -------- | ----------- | ----------------------------------- |
| label | `string` | `undefined` | The name of the statistics display. |
| value | `string` | `undefined` | Statistics show the value.          |

## Slots

| Name    | Parameters | Description   |
| ------- | ---------- | ------------- |
| default | `()`       | Value slot.   |
| label   | `()`       | Label slot.   |
| prefix  | `()`       | Value prefix. |
| suffix  | `()`       | Value suffix. |
