# 可清除

让输入值可以清除（当有值的时候）。

```html
<n-space vertical>
  <n-input type="input" placeholder="可以清除" clearable autocomplete="off" />
  <n-input
    type="password"
    placeholder="可以清除"
    clearable
    autocomplete="off"
  />
  <n-input
    type="textarea"
    placeholder="可以清除"
    round
    clearable
    autocomplete="off"
  />
</n-space>
```
