# Set Attributes on Input Element

```html
<n-space vertical>
  <n-input
    type="input"
    placeholder="Pass autocomplete 'off'"
    :input-props="{ autocomplete: 'off' }"
  />
  <n-input
    type="textarea"
    placeholder="Pass autocomplete 'off'"
    :input-props="{ autocomplete: 'off' }"
  />
</n-space>
```
